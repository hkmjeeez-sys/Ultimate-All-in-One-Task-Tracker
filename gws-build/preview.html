          This is an interactive preview of every visible tab: edit any cell on Master Tasks and the Dashboard, Kanban Board, Monthly/Weekly Calendar, Gantt Chart, Decision Matrix, and Analytics all recompute instantly — mirroring exactly how the COUNTIF/COUNTIFS/FILTER/VLOOKUP formulas in gws-build/scripts/ would behave in the real Google Sheet. It seeds from the same 35-row dataset as 03_master_tasks_values.sh plus the 5 recurring templates from 04_recurring_setup_values.sh (dates anchored to Jul 4, 2026) and computes "today" from your actual clock, so overdue/due-today/weekly-bucket/next-due logic shifts correctly day to day, just like TODAY() would. Reference Data and Recurring Engine stay dimmed since they're hidden helper sheets in the real workbook too. Per-slice donut and single-series bar colors are approximated here since the Sheets API itself can't set those.
